# Resume-Builder-App

**Resume-Builder-App** is a full-stack resume/CV builder designed to help users create polished, customizable resumes in minutes. The app provides a smooth, fast, and scalable experience with modern features.

---

## Features

- Create and customize resumes/CVs easily
- Extract text from PDF files
- Elegant notifications with React Hot Toast
- Smooth navigation using React Router DOM
- State management with React Redux and @reduxjs/toolkit
- Styled with Tailwind CSS and modern icons via lucide-react
- Fast and optimized builds with Vite

---

## Tech Stack

**Frontend:**
- React
- React Router DOM
- React Redux & @reduxjs/toolkit
- React Hot Toast
- react-pdftotext
- Axios
- Tailwind CSS & @tailwindcss/vite
- lucide-react
- Vite & @vitejs/plugin-react

**Code Quality & Development Tools:**
- ESLint with @eslint/js

- eslint-plugin-react-refresh
- TypeScript type definitions
